# 🌙 Tokyo Night Dynamic Clock - Wallpaper Engine

[Portuguese version below]

This is a dynamic wallpaper for **Wallpaper Engine** that simulates a JavaScript/JSON object. It uses the **Tokyo Night** color scheme, offering a modern and minimalist aesthetic for developers and tech enthusiasts.

## 🚀 Technologies Used
- **HTML5**: Semantic structure.
- **CSS3**: Styling with variables, neon glow effects, and animations.
- **JavaScript**: DOM manipulation and real-time clock logic.

## 🛠️ Features
- **Real-Time Monitor**: Displays hours, minutes, and seconds.
- **Dynamic Date**: Automatic updates for day, month, and year.
- **Dev Aesthetic**: Colors based on the popular Tokyo Night theme with an absolute black background (#000000).
- **Pulsing Effect**: Subtle animation on the initial comment to simulate system activity.

## 💻 How to Use

### Option 1: Via Steam Workshop (Recommended)
1. Open **Wallpaper Engine**.
2. Go to the **Workshop** tab.
3. In the search bar, type the name you used for the project (e.g., `Tokyo Night Code Clock`).
4. Click **Subscribe** and apply the wallpaper.

### Option 2: Manual (Local File)
1. Download the `index.html` file from this repository.
2. In Wallpaper Engine, click **Create Wallpaper**.
3. Drag and drop the `index.html` file into the editor window.
4. Save and apply.

# 🌙 Tokyo Night Dynamic Clock - Wallpaper Engine (Português)

Este é um papel de parede dinâmico para o **Wallpaper Engine** que simula um objeto JavaScript/JSON. Ele utiliza o esquema de cores **Tokyo Night**, oferecendo uma estética moderna e minimalista para desenvolvedores e entusiastas de tecnologia.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação semântica.
- **CSS3**: Estilização com variáveis, efeitos de glow neon e animações.
- **JavaScript**: Lógica de manipulação do DOM e atualização em tempo real.

## 🛠️ Funcionalidades
- **Monitor em Tempo Real**: Exibição de horas, minutos e segundos.
- **Data Dinâmica**: Atualização automática de dia, mês e ano.
- **Estética Dev**: Cores baseadas no popular tema Tokyo Night com fundo preto absoluto (#000000).
- **Efeito Pulsante**: Animação sutil no comentário inicial para simular atividade de sistema.

## 💻 Como Usar

### Opção 1: Via Steam Workshop (Recomendado)
1. Abra o **Wallpaper Engine**.
2. Vá na aba **Oficina** (Workshop).
3. No campo de busca, digite o nome que você definiu para o projeto (ex: `Tokyo Night Code Clock`).
4. Clique em **Inscrever-se** e aplique o wallpaper.

### Opção 2: Manual (Arquivo Local)
1. Faça o download do arquivo `index.html` deste repositório.
2. No Wallpaper Engine, clique em **Criar Papel de Parede**.
3. Arraste o arquivo `index.html` para a janela do editor.
4. Salve e aplique.

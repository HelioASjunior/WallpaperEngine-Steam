## 🇧🇷 Português

Este repositório reúne **todos os wallpapers** que desenvolvi e que estão disponíveis na **Oficina do Wallpaper Engine na Steam**. Cada wallpaper é construído com tecnologias web (HTML, CSS e JavaScript), trazendo animações e estéticas únicas para o seu desktop.

Com o tempo, novos wallpapers serão adicionados ao repositório.

---

### 💻 Como Usar

#### Opção 1: Via Steam Workshop (Recomendado)
1. Abra o **Wallpaper Engine**.
2. Vá na aba **Oficina** (Workshop).
3. Pesquise pelo nome do wallpaper desejado.
4. Clique em **Inscrever-se** e aplique o wallpaper.

#### Opção 2: Manual (Arquivo Local)
1. Faça o download do arquivo `.html` desejado neste repositório (pasta `Wallpapers/`).
2. No Wallpaper Engine, clique em **Criar Papel de Parede**.
3. Arraste o arquivo `.html` para a janela do editor.
4. Salve e aplique.

This repository contains **all the wallpapers** I have created and that are available on the **Wallpaper Engine Workshop on Steam**. Each wallpaper is built with web technologies (HTML, CSS, and JavaScript), bringing unique animations and aesthetics to your desktop.

---

### 💻 How to Use

#### Option 1: Via Steam Workshop (Recommended)
1. Open **Wallpaper Engine**.
2. Go to the **Workshop** tab.
3. Search for the wallpaper name.
4. Click **Subscribe** and apply the wallpaper.

#### Option 2: Manual (Local File)
1. Download the desired `.html` file from this repository (inside the `Wallpapers/` folder).
2. In Wallpaper Engine, click **Create Wallpaper**.
3. Drag and drop the `.html` file into the editor window.
4. Save and apply.
